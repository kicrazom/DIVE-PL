# data/ — governance (RODO/GDPR)

**No raw or identifiable participant data is stored in this repository. Ever.**

- Raw/processed data live outside git (see `../.gitignore`).
- Only aggregated, de-identified summaries may feed the public dashboard.
- Anonymisation/pseudonymisation is applied before any analysis export.
- Consent and the RODO information sheet are archived as documents (text), not
  as respondent records.
- Lawful basis, retention period, and data-controller details live in the study
  protocol / OSF project, not here.
