Pipeline

Four AI components in a reviewed loop, governed end-to-end by human oversight and an open-science layer.

1 · LLM agent (Claude / Cowork) — design & orchestration
2 · LLM reviewer (GPT, LLM-as-judge) — validates items, loops fixes back
3 · REDCap — validated instrument, consent / RODO, de-identified capture
4 · AI-built dashboards (Python / React) — aggregated analytics
Open-science layer: GitHub · Zenodo DOI · OSF preregistration · GitHub Pages

Reviewed loop. Stage 1 drafts items; Stage 2 returns structured flags (bias, clarity, validity); the agent revises; a human expert approves before anything reaches REDCap.

Governance. A domain expert governs all four stages. AI augments and is verified; it does not replace expert judgment.
